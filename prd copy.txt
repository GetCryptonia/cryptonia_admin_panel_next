New Admin Panel PRD

1. Dashboard

- View trading volume in a chart, adjust dates (today, last 7 days, last 30 days and custom range) 
- See total volume traded, number of transactions and number of new customers
- See distribution of for each token type/ currency traded in
- View other analytic data for selected date as well, including:
 + Total customers, Completed kyc, completed transactions, total reward spins, referred users
 + Total users that referred users, Reward spins performed, Total transactions that get a reward

2. Balances
- See wallet balance distributions
 + NGN Wallet balance
 + See usd wallet balance and distribution (crypto)
 + See other wallet balances (Usd/euro/gbp) for global payouts
- Withdraw from crypto balance

3. Setup
- Adjust rates (Deposit and withdrawal rates) for each currency
 + usd, eur, gbp, ngn rates as well as an option to set whether it should automatically adjust rates
- Set transaction fees

4. Rewards
- See list of reward types, filter reward types
- Create Rewards

5. customers
- See table of customers (username email, first middle last names, phone number, date joined, last login)
- search by username, first name, last name, phone number, middlename
- Expand a customer to see additional info
 + Added bank accounts
 + Kyc levels completed
 + Last 5 transactions
 + Est. total volume done (might be excluded) Inflow/outflow, wallet balance
 + Block customer
 * Kyc breakdown

6. Transactions
- Search for a transaction by id, session id, transaction hash etc
- See transactions in a table (Account name, transaction id, amount (ngn and usd), transactions status, transaction date, order type)
 + Order types includes:   BUY, SELL, DEPOSIT, WITHDRAW, GLOBAL_PAYOUT, TAG_SEND, TAG_RECEIVE, CARD_FUNDING, CARD_CHARGE, CARD_WITHDRAWAL
- See order info breakdown (See admin panel and app for transaction details)
- Mark a pending/received order as completed by providing session id/hash
- Reverse an existing order and credit user wallet (In case nibbss reverses a transaction, only for received transactions)

7. Members
- See members of the admin panel 
- Adjust the role of an existing members
- Remove member from admin panel

8. Wallet
- See all wallets in table (wallet id, balance, last token traded, tag, last transaction date, potentially total inflow/outflow)
- Search and filter
- Expand a wallet to see information like:
 + Last 10 Transactions
 + Basic user information
 + User id to copy and check in customers table
 + Sort wallets by date created desc/asc, amount desc/asc, and last transaction date desc/asc
